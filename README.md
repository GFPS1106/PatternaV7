# Patterna

Patterna is a shape-agnostic image-analysis pipeline for spatial quantification of immunofluorescence patterns in micropatterned pluripotent stem-cell colonies and 2D gastruloid-like systems.

The pipeline converts each colony into a normalized spatial coordinate system using a spiderweb sampling strategy. Intensities are sampled from the colony centroid to the true colony boundary across many angular directions, allowing marker organization to be quantified from center to edge independently of colony shape.

Patterna is designed to work with circular, oval, triangular, tear-shaped, and irregular micropatterns without relying on manually selected one-dimensional line profiles.

## Main features

* Colony masking and QC overlays
* Shape-agnostic spiderweb sampling
* Channel × radius × angle intensity grids
* Radial intensity profiles for each marker
* Per-channel max-normalized radial profiles
* Rectangular and polar topography maps
* Composite topography maps
* Orthogonal major/minor axis profiles
* Anisotropy and spatial fingerprint metrics
* Grouped radial profiles across multiple micropatterns
* Grouped heatmaps and grouped polar maps
* Single-cell/nuclei-level marker quantification
* StarDist-based nuclei segmentation
* Optional hybrid dense-watershed fallback
* Configurable cell phenotype classification
* Cell-intensity maps and phenotype maps
* Integrated spatial summary panels

## Input

Patterna expects microscopy images in OME-TIFF or TIFF format.

The current default channel mapping is:

```text
clean ch0 = DAPI
clean ch1 = TBXT
clean ch2 = SOX2
clean ch3 = GATA6
```

If a 5-channel image is detected, the script removes the brightfield channel and keeps the cleaned fluorescence stack for downstream analysis.

## Quick start

1. Clone or download this repository.
2. Install the required Python packages.
3. Place test images inside the `test_data/` folder.
4. Open `src/PatternaV7.py`.
5. Edit the input and output paths:

```python
INPUT_DIR = r"path/to/test_data"
OUT_DIR = r"path/to/Patterna_output"
```

6. Run the script:

```bash
python src/PatternaV7.py
```

Outputs will be saved in the folder defined by `OUT_DIR`.

## Example data

This repository includes a small test image in `test_data/`.

The `example_outputs/` folder contains the output generated by running Patterna on the test image. This folder can be used to check whether a new run produced the expected output structure.

## Notes on phenotype classification

Patterna can classify cells as marker-positive or marker-negative using configurable thresholds. The current thresholding options are intended for exploratory analysis and should be calibrated with appropriate biological controls before final interpretation.

Percentile-based thresholds are useful for testing and visualization, but they should not be treated as final biological ground truth unless validated.

## Notes on nuclei segmentation

Patterna uses StarDist for nuclei segmentation by default. For dense colonies where StarDist under-segments, the hybrid mode can fall back to a dense watershed segmentation approach.

Segmentation quality depends on DAPI signal quality, colony density, image resolution, and the selected parameters. Always inspect the QC overlays before interpreting single-cell outputs.

StarDist and TensorFlow are required when `NUC_SEGMENTATION_MODE = "stardist"` or `"hybrid"`.
